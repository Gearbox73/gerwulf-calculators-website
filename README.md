# Gerwulf BCBC Spatial Separation Calculator

Public web interface for the BCBC Spatial Separation and Unprotected Opening Calculator.

## Live Site
https://gerwulf.co/spatial-separation-calculator-bcbc/

## Features
- Spatial separation calculations per BCBC 9.10.14.4 and 9.10.15.4
- Unprotected opening calculations
- Interpolation and tabular methods
- Multi-wall face project management
- PDF export and clipboard copy

## Technical Stack
- Pure JavaScript (no frameworks)
- Azure-hosted calculation API
- GitHub Pages hosting
